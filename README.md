5. 注意事项

内存：大图片可能需要较多内存
GPU 支持：如果需要更快处理，可以启用 GPU
中文支持：默认支持中文，如果需要其他语言可以调整配置
这个方案是完全开源的，可以在本地部署，不需要任何付费 API。表格识别准确率取决于图片质量，建议使用清晰的表格截图或扫描件
